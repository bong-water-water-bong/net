## what is net?

net has her finger on the pulse of every connection. she pings the router, watches the wireguard tunnel, discovers every device on the lan, and flags anything that doesn't belong. she's the family's nervous system — if a cable gets unplugged three rooms away, she feels it.

### family

net is one of the reflex group — the younger siblings in the ai family, working under their eldest brother [meek](https://github.com/bong-water-water-bong/meek). their father [halo ai](https://github.com/bong-water-water-bong/halo-ai) built the stack. their mother [echo](https://github.com/bong-water-water-bong/echo) speaks for the family. net just does the work.

## quick start

```bash
# run as part of meek (recommended)
meek scan --agent net

# standalone
python net.py scan
python net.py status
```

## schedule

**every 5 min** — runs automatically via meek's systemd timers.
